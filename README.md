# Usage
```bash
pip install -r requirements.txt
python main.py
```

# Config
example `config.json`
```json
{
  "siteConfig": {
    "savePath": "codes",
    "scheme": YOUR_SCHEME,
    "host": YOUR_HOST,
    "ignoreExisted": true,
    "ignoreZero": true,
    "limitGold": 0,
    "keepGold": false
  },
  "username": YOUR_USERNAME,
  "password": YOUR_PASSWORD
}
```
